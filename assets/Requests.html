Python 'requests' Module for Beginners

The 'requests' module in Python allows you to send HTTP requests to web services and retrieve data from websites.

Installation

You can install the 'requests' module using pip:

pip install requests

Basic Usage

To send a GET request and retrieve content from a website:

import requests

response = requests.get('https://www.example.com')
print(response.text)

Here, we import the 'requests' module, send a GET request to 'https://www.example.com', and print the response content.

HTTP Methods

The 'requests' module supports various HTTP methods:

  • GET: Retrieve data from a server.
  • POST: Send data to a server.
  • PUT: Update data on a server.
  • DELETE: Remove data from a server.

Handling Response

You can access response data using the 'response' object:

import requests

response = requests.get('https://www.example.com')

# Status code
print(response.status_code)

# Response content
print(response.text)

# Headers
print(response.headers)

The 'status_code' attribute contains the HTTP status code, 'text' contains the response content, and 'headers' contains the HTTP headers.

Headers and Parameters

You can set custom headers and parameters in your requests:

import requests

headers = {'User-Agent': 'My User Agent'}
params = {'key': 'value'}

response = requests.get('https://www.example.com', headers=headers, params=params)

Error Handling

You should handle exceptions when making requests:

import requests

try:
    response = requests.get('https://www.example.com')
    response.raise_for_status()  # Raise an exception for HTTP errors
except requests.exceptions.HTTPError as err:
    print(f"HTTP error: {err}")
except requests.exceptions.RequestException as err:
    print(f"Request error: {err}")

Conclusion

The 'requests' module is a powerful tool for working with HTTP requests in Python. It allows you to interact with web services, retrieve data, and handle various aspects of HTTP communication.
